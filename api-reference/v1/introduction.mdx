# Getting Started

Welcome to the Metafy API documentation! Here you will find everything you need to get started with the Metafy API.

<Steps titleSize="h3">
  <Step title="Create an Account">
    Make sure you have a [Metafy account](https://metafy.gg).
  </Step>
  <Step title="Become a Metafy Partner">
    <Warning>
      Currently, we only allow our Partners to have access to our API.

      This will change very soon.
    </Warning>

    Become a [Metafy Partner](https://metafy.gg/pro) to get access to the API.
  </Step>
  <Step title="Start building">
    Start building your application and check out the [links below](#find-support) for support.
  </Step>
</Steps>
